This is just a demo to show how fast search can happen. Obviously not what you'd really search on or how you want it to look.

You'd have the data in google sheets and export to a CSV on your hard drive. A 'utility script' would sit in the same folder as the CSV files and you could run it anytime to generate a new index.html page that sucked in all the CSV data and made embedded arrays with it. Then when you were ready, you'd replace the index.html file on your GitHub account with the new index.html file. That way you don't have to worry about cutting-and-pasting data into the index file (it would all be done automatically).
